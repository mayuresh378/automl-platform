## 📌 Overview

AutoML Platform is designed to make machine learning accessible through an interactive dashboard that enables users to:

- Upload datasets
- Perform preprocessing
- Train ML models
- Compare model performance
- Visualize results
- Deploy trained models
- Monitor predictions

The platform provides an elegant enterprise-grade user experience inspired by modern SaaS products.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/mayuresh378/automl-platform.git
```

```
cd automl-platform
```

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on

```
http://localhost:5173
```

---

## Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```
